# final_project_data_loves

Short description: Python-based analysis pipeline for coupon redemptions and household transactions. Cleans and merges coupon, transaction, product and demographic data; produces aggregate tables, cohort and campaign analyses, basic visualizations, and exports cleaned datasets for BI (Tableau).

Goals: reproduce the analyses from the dashboard, compute campaign-level KPIs, produce cohorts and household metrics, and generate visuals for reporting.
